===============================================
EMAILJS QUICK START GUIDE
===============================================

WHAT IT DOES:
- Sends email when timer STARTS
- Sends email when timer ENDS
- Automatically to: Vishal165610@gmail.com
- Completely automatic, no user input needed

===============================================
5 MINUTE SETUP
===============================================

STEP 1: Go to https://www.emailjs.com
        Sign up free (takes 1 minute)

STEP 2: Copy your User ID from Dashboard
        (top left corner)

STEP 3: Create Gmail service
        - Go to Email Services
        - Click "Create New Service"
        - Select Gmail
        - Connect Vishal165610@gmail.com
        - Copy Service ID

STEP 4: Create 2 Email Templates
        - Template 1: Sleep Start
        - Template 2: Sleep End
        (Use HTML provided in EMAILJS_SETUP_COMPLETE.md)

STEP 5: Update emailjs-config.js
        Replace these 4 values:
        - USER_ID
        - SERVICE_ID
        - TEMPLATE_START_ID
        - TEMPLATE_END_ID

DONE! 🎉

===============================================
QUICK TROUBLESHOOTING
===============================================

Emails not sending?
1. Check User ID in emailjs-config.js
2. Check Service ID in emailjs-config.js
3. Check Template IDs in emailjs-config.js
4. Check email in browser console (F12)
5. Check EmailJS activity log

Still not working?
- Verify Gmail password generated correctly
- Check that Vishal165610@gmail.com is connected
- Read EMAILJS_SETUP_COMPLETE.md for details

===============================================

{{sleep_end_time}} = When sleep ended
