# Sweet Dreams - Sleep Tracker App
## ✨ Latest Updates - Dark Theme + Real-time Live Timer

### 🌙 New Features

#### 1. **Dark Theme with Yellow Accents**
- Complete dark background (`#0f0f0f` to `#1a1a1a`)
- Beautiful yellow primary color (`#fbbf24` with `#f59e0b` variants)
- Glowing text effects on titles for that elegant look
- All colors carefully chosen for contrast and readability

#### 2. **Real-Time Live Notifications**
When the timer starts:
- Floating notification appears in the top-right corner
- Shows "She is sleeping now! 😴" with live timer
- Updates every second showing exact sleep duration
- Animated slide-in effect with smooth transitions
- Disappears when timer stops

#### 3. **Enhanced Timer Display**
- Large glowing timer with pulse animation
- Status badge shows "SLEEPING NOW" when active
- Yellow color with text-shadow glow effect
- Real-time updates sync across all pages

#### 4. **Fully Responsive Design**
- Mobile-first design approach
- Perfect on Android phones, tablets, and desktops
- Flexible navbar with wrapped elements
- Touch-friendly button sizing
- Smooth animations on all screen sizes

- Automatic redirect to dashboard if already logged in
- Secure logout option always available
